# Failure scenarios: portfolio modernization roadmap (EA)

- **Big-bang program** — no incremental value; **funding** pulled mid-flight; **milestone** delivery fails.
- **Wrong sequencing** — modernizing **UX** before **data** or **identity** — systems still cannot integrate.
- **Vendor lock-in** without exit — contract and **portability** risk; **interface** ownership inside enterprise.
- **Security / compliance** late engagement — **blocker** at go-live; **shift-left** reviews on each wave.
- **Operating model gap** — new platform built but **no** SRE/cost ownership; **reliability** regresses.
- **Stakeholder misalignment** on **NFRs** — “done” on features but **not** on SLOs or DR.

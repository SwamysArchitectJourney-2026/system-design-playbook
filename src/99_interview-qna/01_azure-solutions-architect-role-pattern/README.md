# Azure Solutions Architect — interview track

This folder is structured for **execution**, not browsing: templates, repeatable question packs (answer + diagram + trade-offs + follow-ups), Azure-specific depth, and cross-domain scenarios.

## Layout

| Path | Purpose |
|------|---------|
| `01_templates/` | Spine, trade-off patterns, follow-up checklist (Azure-flavored) |
| `02_core_questions/` | Canonical designs you can narrate in 35–45 minutes |
| `03_azure_specific/` | Service-composition patterns common in Azure loops |
| `04_cross_domain/` | AI, payments, and other integrations on Azure |

## How to practice

1. Read `01_templates/system-design-template.md` (links to shared spine in `interview-execution/`).
2. Pick one `02_core_questions/*/answer.md` and **speak it out loud** without reading.
3. Drill `followups.md` for the same question until answers are under two minutes each.

Shared grill and golden flow: `../interview-execution/`.
